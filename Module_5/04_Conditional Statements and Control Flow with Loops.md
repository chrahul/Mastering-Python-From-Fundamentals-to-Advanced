### **Main Content**

#### **What Is Control Flow**

So far, all the programs you have written run line by line, top to bottom.
This is called linear execution.

But real programs rarely work that way.

Sometimes you need to:

* Repeat a block of code
* Stop execution based on a condition
* Skip certain steps and continue

This decision making and repetition together is called **control flow**.

---

#### **Why Loops Matter So Much**

Loops allow us to repeat statements without writing them again and again.

For example:

* Reading multiple inputs
* Processing a list of values
* Running calculations repeatedly
* Iterating over data in data science and machine learning

Almost every real program uses loops in some form.

That is why loops are considered one of the most important and challenging topics in programming.

---

#### **Loops in Python**

Python provides two main types of loops:

* **while loop**
* **for loop**

Both are powerful, but they are used in slightly different situations.

You will also learn supporting keywords that control loop behavior:

* break
* continue
* pass

These keywords help you control when a loop should stop, skip, or do nothing.
